## Featured Project

### 🔗 [Linkly](https://github.com/adocxwork/linkly) — Full-Stack Link-in-Bio Platform
[![Live](https://img.shields.io/badge/Live-success?style=flat-square&logo=vercel)](https://linkly-plum.vercel.app/) [![Spring Boot](https://img.shields.io/badge/Spring_Boot-3-6DB33F?style=flat-square&logo=spring)](https://github.com/adocxwork/linkly) [![React](https://img.shields.io/badge/React-18-61DAFB?style=flat-square&logo=react)](https://github.com/adocxwork/linkly) [![PostgreSQL](https://img.shields.io/badge/PostgreSQL-16-336791?style=flat-square&logo=postgresql)](https://github.com/adocxwork/linkly)

Deployed end-to-end on Vercel + Render. Lets creators consolidate their digital presence into one customizable public profile — with real-time click analytics, drag-and-drop link reordering, UPI-based payment collection, a public messaging inbox, and a role-based admin dashboard. Secured with stateless JWT authentication via Spring Security.

**[Live Demo →](https://linkly-plum.vercel.app/)** &nbsp;·&nbsp; **[Source →](https://github.com/adocxwork/linkly)**
